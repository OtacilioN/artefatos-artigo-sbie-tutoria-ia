# Derived analysis data

- `da_artifacts/`: fixed dialogue-act sequences, global axes, Markov cluster
  assignments, and cluster-average matrices.
- `process_mining_outputs/`: structural metrics and dominant transitions used
  to characterize the six second-order profiles.
- `trajectory/`: student-by-assignment profile matrix, trajectory groups, and
  transitions between consecutive assignments.

The analytical unit is a pseudonymous student-assignment pair. The fixed input
contains 916 units, 1,727 dialogue traces, 16,364 dialogue-act events, 29
observed states, and 276 observed second-order superstates.

No message text or tutor response is distributed in this repository. See
`DATA_NOTICE.md` for provenance and reuse information.
